1. Abrir los proyectos por separado

Abrir client y server en ventanas distintas del editor.
Cada uno se levanta por separado.

2. Instalar los requerimientos del servidor (Python)

En el entorno virtual del proyecto server, ejecutar:

pip install -r requirements.txt

3. Crear y aplicar las migraciones

python manage.py makemigrations
python manage.py migrate

4. Crear usuarios iniciales
Ejecutar:

python manage.py create_default_users

Esto crea dos usuarios en la base de datos:
Administrador
Email: admin@local
Clave: admin123

Colaborador
Email: colab@local
Clave: colab123

5. Levantar el servidor
python manage.py runserver

Para levanta el el cliente

npm run dev
